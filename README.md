# LinguaPlay

Plataforma educacional gamificada para o ensino de inglês e espanhol para crianças do ensino infantil e fundamental, com mini games e área do professor para acompanhamento de dados.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Como rodar localmente

Requer Node.js e npm instalados.

```sh
# 1. Clone o repositório
git clone <YOUR_GIT_URL>

# 2. Entre na pasta
cd <YOUR_PROJECT_NAME>

# 3. Instale as dependências
npm i

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

## Rotas principais

- `/` — Landing page institucional
- `/game` — Mini game educacional ("Palavra Mágica")
- `/professor` — Área do professor com dashboard de dados
